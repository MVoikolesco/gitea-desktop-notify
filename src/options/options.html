Como gerar um token?
Mínimo: 0.5 minutos (30 segundos)

ℹ️ Como usar:

  1. Configure a URL do seu servidor Gitea
  2. Gere um token de acesso na sua conta Gitea
  3. Cole o token no campo acima
  4. Clique em "Testar Conexão" para verificar
  5. Clique em "Salvar" para aplicar as configurações
